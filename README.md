# BeautyShop #

This is an online beauty products shopping application.

### What is this repository for? ###

* BeautyShop Ecom
* Version: 1.0

## Getting Started

### Tech Stack ###
* Vite + React + Typescript
* React Router v7.0+
* Sass/scss
* Tailwindcss 4.0+
* Redux Toolkit
* Axios

### Prerequisites

- Node.js 21+
- npm 9+

### Installation

```bash
# Install dependencies
npm install

# Start development servers
npm run dev

#Build for development
npm run build:dev

# Build for production
npm run build:prod

# Install Serve globally
npm i --global serve

# Serve production build
npm run serve:prod
```
